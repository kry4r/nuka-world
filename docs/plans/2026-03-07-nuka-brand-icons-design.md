# Nuka Brand Icons Design

**Date:** 2026-03-07

## Goal

Replace the current in-app `NukaLogo` with a brand mark that matches the reference images in `docs/logo`, while also creating a matching lockup that includes the `Nuka` wordmark.

## Approved Direction

Two brand families will be drawn:

- `A / natural` ¡ª softer mountain silhouette inspired by `docs/logo/logo.png`
- `B / geometric` ¡ª stronger dual-peak mark inspired by `docs/logo/Fd_GX65wSMa_Zke1iWkWMA.jpg`

Each family includes:

- a pure `mark`
- a `lockup` with the `Nuka` wordmark

## Product Use

- `NukaLogo` becomes the default in-app mark and should use the `geometric` family by default
- `NukaLockup` is added as a reusable brand asset for future hero/settings use
- both assets remain tintable via `currentColor`

## Visual Rules

- no bitmap tracing
- clean SVG geometry
- small-size readability first
- warm desktop UI compatibility
- no heavy color dependence in the default implementation
